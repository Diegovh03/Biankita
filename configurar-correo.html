Configurar correo (solo una vez)

FormSubmit solo te manda avisos a ti. Con EmailJS el enlace si llega a la otra persona.

  1. Entra a emailjs.com y crea cuenta gratis (con tu Gmail).
  2. Email Services → Add New Service → elige Gmail → conecta daaron.valdiviah@gmail.com.
  3. Email Templates → Create Template:
    • To Email: {{to_email}}
    • Subject: Tu carta te espera
    • Content:
    Hola,

Te envio el enlace de tu carta:

{{link}}

Abrelo desde tu laptop para que se vea mejor.

Con carino,
Diego
  4. En Account → API Keys copia tu Public Key.
  5. Abre el archivo js/email-config.js en tu carpeta Biankita y pega los 3 datos:
window.EMAILJS_CONFIG = {
  publicKey: "tu_public_key",
  serviceId: "service_xxxxx",
  templateId: "template_xxxxx"
};

Luego sube a GitHub:

git add js/email-config.js
git commit -m "Configurar EmailJS"
git push

Como probar

  1. Usa un correo distinto al tuyo (Outlook, iCloud, el de Biankita).
  2. Abre acceso.html, pon ese correo y envia.
  3. Revisa ese correo (y spam). Debe llegar con is.gd/OBRhL8.
  4. Lo que tu ves de FormSubmit ("Someone submitted your form") es solo aviso para ti, no es el enlace.
