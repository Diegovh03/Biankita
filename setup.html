Conectar tu Gmail (una sola vez)

Una pagina web no puede mandar correos sola. Hay que conectar tu Gmail una vez. Despues de eso: alguien pone su correo → le llega el enlace. Automatico.

Paso 1 — Abrir Google Script

Clic aqui: crear proyecto nuevo

Paso 2 — Pegar este codigo

Borra todo lo que hay en el editor y pega esto:

var CARTA_URL = "https://is.gd/OBRhL8";
var ENVIADO_URL = "https://diegovh03.github.io/Biankita/enviado.html";

function doPost(e) {
  var email = (e.parameter.email || "").trim().toLowerCase();
  if (!email || !/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email)) {
    return HtmlService.createHtmlOutput("Correo invalido.");
  }
  GmailApp.sendEmail(
    email,
    "Tu carta te espera",
    "Hola,\n\nTe envio el enlace de tu carta:\n\n" + CARTA_URL +
    "\n\nAbrelo desde tu laptop.\n\nCon cariño,\nDiego"
  );
  return HtmlService.createHtmlOutput(
    '<meta http-equiv="refresh" content="0;url=' + ENVIADO_URL + '">'
  );
}

Paso 3 — Publicar

  1. Arriba: Implementar → Nueva implementacion
  2. Tipo: Aplicacion web
  3. Ejecutar como: Yo (daaron.valdiviah@gmail.com)
  4. Acceso: Cualquier persona
  5. Implementar → Google pedira permiso para enviar correos → Permitir
  6. Copia la URL que termina en /exec

Paso 4 — Pegar la URL aqui y en el proyecto


Pega esa URL en el archivo js/google-config.js:

window.GOOGLE_SCRIPT_URL = "PEGA_LA_URL_AQUI";
