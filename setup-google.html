Envio automatico por Gmail

FormSubmit no manda el enlace a la otra persona. Con Google Script si (gratis, 5 min).

  1. Abre script.google.com → proyecto nuevo.
  2. Borra todo y pega el codigo del archivo google-apps-script/enviar-enlace.gs de tu carpeta Biankita.
  3. Guardar → Implementar → Nueva implementacion → tipo Aplicacion web.
  4. Ejecutar como: Yo · Acceso: Cualquier persona → Implementar.
  5. Copia la URL que termina en /exec.
  6. Abre js/google-config.js y pegala:
window.GOOGLE_SCRIPT_URL = "https://script.google.com/macros/s/XXXX/exec";

Sube a GitHub:

git add js/google-config.js
git commit -m "Activar Gmail automatico"
git push

Mientras tanto (sin configurar)

La pagina abre el correo de la persona con el enlace ya escrito. Ella pulsa Enviar y le llega a su bandeja.
